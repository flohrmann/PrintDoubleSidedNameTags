excel row transformer for double sided printing

how it works:

this will help you generate front/back print layouts for name tags <3

example input (.xml):
Fani (1)
Mara (2)
Vlad (3)
Alex (4)

example output (.xml):
Fani (1)
Mara (2)
Vlad (3)
Alex (4)
Mara (2)
Fani (1)
Alex (4)
Vlad (3)

if you were to print it would look like:

front
Fani (1) Mara (2)
Vlad (3) Alex (4)
back
Mara (2) Fani (1)
Alex (4) Vlad (3)
