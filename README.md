# PrintDoubleSidedNameTags (name tag printer tool)
- made originally for maria to  make her life easier
- based on a name list gives back a name list thats optimized for printing double sided name tags

- this is a simple browser-based tool that lets you drag and drop an excel file, transform its rows using predefined patterns, and preview/export the result for name tag printing <3


## ~ features ~
- drag & drop excel (.xlsx) files
- reads spreadsheet directly in the browser
- transforms rows using custom patterns (“folge”)
- supports different chunk sizes (e.g. 4 or 8 rows)
- live table preview of original and transformed data
- export transformed data as a new excel file (.xlsx)


## ~ how it works ~

### download
1. click on the file (GetDoubleSidedNameTags.html)
2. click **download raw file** (or “download”)
3. save it to your computer

### run locally
1. go into the folder you safed the file into
2. double-click the HTML file (GetDoubleSidedNameTags.html)
--> this will open in your browser (Chrome, Firefox, Edge, etc.)

### using the tool
1. select a transformation pattern (e.g. 4-row or 8-row mode)
2. drag and drop an .xlsx file into the drop zone
--> view: original table & transformed table
3. click "download XLSX" to export the result
